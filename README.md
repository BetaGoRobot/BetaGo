> 正在尝试实现的开黑啦机器人

出于娱乐的意义，机器人名称BetaGo意为AlphaGo的迭代

基于GO的开黑啦机器人SDK `khl` 开发，目标是开发一套有丰富功能的开黑啦通用机器人框架

## Current Features:

    [x] GateWay转发Type字段的映射配置

    [x] 自动at回复

    [x] 字段简单过滤
